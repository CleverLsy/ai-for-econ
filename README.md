# AI for Econ

> 经济学研究者的 AI 协作手册

📖 在线阅读：https://cleverlsy.github.io/ai-for-econ/

ps：最初的设计思路是三个模块——
  > - 模块一：基本概念，对一些术语建立直觉认知；
  > - 模块二：工具栈，从底层大模型到 Agent，到 skill 集合站、MCP 工具、常用编辑器等的说明和介绍；
  > - 模块三：具体研究场景中的应用，从选题、文献综述，到数据分析、审稿，再到备课或论文 slides 制作等。
  
  > 这个项目最初是为了满足自己的一些想法和爱好做的探索，目前内容多由 AI 生成，仍在玩具阶段。我想让它的完善过程更自然——边学边用、边记录边输出，以节省时间成本。如果有任何问题或建议，欢迎在 issue 里提出。

## 这是什么

一本写给经济学研究者的 AI 协作手册。从概念辨析、工具选型，到围绕真实研究 Pipeline 的工作流说明，再到一个完整可复现的研究案例。

不是"提示词大全"，而是一套**可以反复使用的方法论**。

## 内容结构

- **概念地图**：LLM、Agent、Harness、RAG、Skill、MCP 的本质
- **工具栈**：按门槛分层，国内外可访问性，学术专用工具
- **研究 Pipeline**：选题→检索→精读→数据→分析→写作→投稿，每个环节的人机协作模式
- **自己写 Skill**：从模仿到创造
- **案例库**：一个真实研究项目从头到尾，全部代码与提示词公开
- **边界与避坑**：学术伦理、数据隐私、AI 幻觉的杀伤力
