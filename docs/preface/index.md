# 写在前面

这本手册最初是一个私人项目——为了满足自己的一些想法和爱好，把平时用 AI 做研究的经验整理成可以复用的东西。

最初的设计思路是三个模块：

- **模块一：基本概念** — 对 LLM、Agent、RAG、Skill、MCP 这些术语建立直觉认知，不被名词淹没
- **模块二：工具栈** — 从底层大模型到 Agent，到 Skill 集合站、MCP 工具、常用编辑器的说明和介绍
- **模块三：具体研究场景** — 从选题、文献综述，到数据分析、审稿，再到备课或论文 slides 制作

现在的结构在这三个模块基础上做了细化，但主线没变。

这个项目目前仍在**玩具阶段**，内容多由人机协同写作完成（见[元说明](meta.md)）。我想让它的完善过程更自然——边学边用、边记录边输出，而不是等到"写完"再发布。如果有任何问题或建议，欢迎在 [GitHub Issue](https://github.com/CleverLsy/ai-for-econ/issues) 里提出。
